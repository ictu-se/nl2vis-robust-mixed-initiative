# Paper 16 Reproducibility Package

Code and experimental inputs for:

`Robust Mixed-Initiative Decision Support for Ambiguous NL2Vis: Candidate Pools, Noisy Feedback, Escalation, and Action Costs`

This package is GitHub-ready and contains:

- code needed to regenerate the main experimental summaries and figures
- the minimal ranked-row JSON inputs required by the scripts
- one fixed controller-summary input reused by the budget sweep
- reference summary outputs for result verification

It intentionally does **not** include the paper manuscript, LaTeX sources, or submission files.

## Repository Layout

- `code/interaction_utils.py`: shared ranking, slot, clarification, and repair utilities
- `code/run_escalation.py`: escalation experiments for ambiguous requests
- `code/run_clarification_robustness.py`: clarification robustness under noisy answers
- `code/run_repair_robustness.py`: repair robustness under noisy edits
- `code/run_budget_sweep.py`: budgeted controller sweep from a fixed controller-summary input
- `code/run_extended_analysis.py`: regime-conditioned analysis and bootstrap confidence intervals
- `code/run_all.py`: one-command reproduction entrypoint
- `code/verify_results.py`: checks generated summary JSON files against included references
- `data/ranked_pools/`: minimal ranked-row experimental inputs
- `data/reference_inputs/controller_summary.json`: fixed controller summary used by the budget sweep
- `reference_results/`: expected summary JSON outputs
- `docs/DATA_DEPENDENCIES.md`: provenance note for included data

## Environment

- Python 3.10+

Install dependencies:

```bash
pip install -r requirements.txt
```

## Reproduce All Results

From the repository root:

```bash
python code/run_all.py
```

Generated outputs will be written to:

- `artifacts_runtime/escalation/`
- `artifacts_runtime/clarification_robustness/`
- `artifacts_runtime/repair_robustness/`
- `artifacts_runtime/budget_sweep/`
- `artifacts_runtime/extended_analysis/`

## Verify Against Reference Outputs

After running all experiments:

```bash
python code/verify_results.py
```

If everything reproduces correctly, the generated summary files should match the included reference summaries.

## Notes

- This package is designed for reproducibility, not for manuscript preparation.
- Figures are regenerated by the scripts; they are not versioned as source assets here.
- The budget-sweep script intentionally reuses a fixed controller summary so the action-cost analysis can be reproduced without rerunning the full controller pipeline.
